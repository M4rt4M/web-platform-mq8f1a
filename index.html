Noughts and Crosses

Your mission

Below is a simple noughts and crosses board which should already let you click on squares in order to make a move - try it out.

Unfortunately it doesn't currently detect who has won, we want you to implement the code to fix that.

Extension Points

It doesn't matter if you don't get here, these are just some extra tasks to consider if you have some spare time.

  • Detect a draw
  • Provide a way to reset the game to its start state
  • Stop players from making extra moves when the game is over
  • Make the game work for a bigger board, say 4x4
